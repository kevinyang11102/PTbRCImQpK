# 内容介绍

本项目是一个宠物爱心组织管理系统，主要功能包括宠物信息管理、领养申请处理、志愿者管理、物资管理等。系统基于 SpringBoot 框架开发，前端采用 JS、Vue 和 CSS3 技术，数据库使用 MySQL。通过本项目，您可以了解到如何运用 Java 和 SpringBoot 快速开发一个实用的管理系统。

# 核心代码

以下是本项目中的一段核心代码，展示了如何使用 SpringBoot 和 MySQL 实现宠物信息查询功能：

```java
// 引入相关依赖
@Autowired
private PetRepository petRepository;

// 查询所有宠物信息
public List<Pet> getAllPets() {
    return petRepository.findAll();
}

// 根据宠物类型查询宠物信息
public List<Pet> getPetsByType(String type) {
    return petRepository.findByType(type);
}
```
